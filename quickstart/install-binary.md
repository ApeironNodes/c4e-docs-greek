# Binary Installation 

## Binaries

Clone and build c4e using `git`:

```bash
git clone https://github.com/chain4energy/c4e-chain.git
cd c4e-chain
git checkout todo
make install
```

Check that the binaries have been successfuly installed:

```bash
c4ed version
```
